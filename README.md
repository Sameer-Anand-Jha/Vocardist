# Vocardist
Just a hobby project to separate lyrical and instrumental stems from music using demucs
Steps to replicate this
#### Download static version of ffmpeg.exe from https://www.ffmpeg.org/download.html, download the Windows one.
#### Download this directory as is, and keep the .exe in the same dir
#### run 'python app.py'

#### Work is underway to make it a standalone Windows executable, but running into issues with demucs while bundling.
